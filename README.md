# Worker Progress Report

# Project Overview

The Worker Progress Report is a web-based application developed using HTML, CSS, and JavaScript. It allows workers to enter their recovery and return-to-work information through a structured multi-page form. After completing the form, the application generates a professional report that can be printed or saved as a PDF.

This project replicates the layout of the Worker Progress Report PDF while providing an interactive and user-friendly interface.

# Features

- Multi-page form (3 pages)
- Worker information collection
- Return-to-work details
- Recovery information
- Pain scale selection
- Medical treatment details
- Medication and home exercise information
- Additional comments section
- Certification before submission
- Automatic report generation
- Print or Save as PDF functionality
- Responsive and clean user interface

# Technologies Used

- HTML5
- CSS3
- JavaScript

# Project Workflow

1. Open the application in a web browser.
2. Enter the worker details.
3. Fill in the Return to Work information.
4. Enter Recovery and Medical details.
5. Complete the certification.
6. Click *Submit Report*.
7. A formatted Worker Progress Report is generated.
8. Print or Save the report as a PDF.
